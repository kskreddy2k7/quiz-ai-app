## â“ Troubleshooting

### PWA Won't Install
- **Chrome:** Make sure you're using the latest version
- **iOS:** Only Safari supports PWA installation on iOS
- **Desktop:** Check if the site is using HTTPS
- Try clearing browser cache and reloading

### APK Installation Blocked
- Enable "Install from Unknown Sources" in Settings
- Make sure the APK downloaded completely
- Try using a different browser to download
- Check available storage space

### App Won't Load
- Check internet connection (required for AI features)
- Try clearing app data/cache
- Reinstall the app
- Make sure you're using a modern browser

### AI Not Working
- Requires internet connection
- Check if the server is online: https://[your-app-url]/health
- Try refreshing the page
- Contact support if issue persists
